# 🎓 Student Pass/Fail Prediction (Machine Learning)

This project uses Machine Learning to predict whether a student will **PASS or FAIL** based on marks.

---

## 🚀 Features

* Simple dataset
* Logistic Regression model
* Accuracy calculation
* User input prediction

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn

---

## ▶️ How to Run

```bash
pip install pandas scikit-learn
python ml_project.py
```

---

## 🎯 Example Output

```
Model Accuracy: 0.85
Enter student marks: 45
Result: PASS
```
